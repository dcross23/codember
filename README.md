# Codember in Python 🗓️ 
Solutions for codember challenges: https://codember.dev/

## Structure 🌳
1 challenge, 1 folder
```
.
├── README.md
└── src
    ├── c1
    │   ├── c1.py
    │   └── input.txt
    ├── c2  
    │   ├── c2.py
    │   └── input.txt
    ├── c3
    │   ├── c3.py
    │   └── input.txt
    └── c4
        └── c4.py
```

## Usage 💾
To run any of the challenges, follow this example:
```
$ cd src/c1/
$ python3 c1.py
```
